👨‍👩‍👧 Cuentas de Apoderados
Los apoderados ingresan con el DNI del alumno como usuario y contraseña inicial. Usa este botón para crear o actualizar las cuentas de todos los alumnos existentes.
⚠️ Si ya existe una cuenta no se modifica. Solo crea las que faltan.
⚠️ Zona de Peligro — Resetear Sistema
Borra permanentemente todos los datos del sistema:
alumnos, registros de asistencia, incidentes, cuentas de apoderados y usuarios del personal.
Solo se conserva el usuario administrador principal.
Esta accion no se puede deshacer.
👻 Cuentas de apoderado huérfanas
Cuentas de Firebase Auth de apoderados cuyos alumnos ya no existen. Deben borrarse manualmente desde la consola de Firebase.
¿Cómo borrarlas?
1. Ve a Firebase Console → Authentication
2. Busca los emails con formato dni@apo.marello.pe
3. Elimina los que correspondan a alumnos ya borrados
